1. Описание проекта

1.1 Проблема

В изолированных контурах (air-gapped environments) отсутствует доступ к внешним сервисам валидации YAML. Существующие инструменты требуют интернет-соединения или сложных зависимостей.

1.2 Решение

Разработан pure Bash валидатор с возможностями:

  1. Валидация Kubernetes манифестов (134 проверки)
  2. Автоматическое исправление распространённых ошибок (13 типов)
  3. Graceful degradation — работа без внешних зависимостей
  4. Security-first подход — защита от Path Traversal, ReDoS, инъекций

1.3 Целевая аудитория

  • DevOps-инженеры в air-gapped окружениях
  • Security-специалисты
  • Разработчики Kubernetes-приложений

  • ✅ Исправление: использование LC_ALL=C

  • ✅ Время: 2 минуты

Без ИИ: ~30-60 минут (debugging + поиск в Google)
